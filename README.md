# E-Commerce Sales Analysis and Business Intelligence Dashboard

---

## Project Overview

This project is a complete end-to-end data analytics and machine learning solution for an e-commerce business. It includes data generation, cleaning, exploratory data analysis, feature creation, visualization, dashboard development, and predictive modeling.

---

## Objective

To analyze e-commerce sales data and generate actionable business insights such as:
- Customer behavior analysis
- Product performance tracking
- Revenue and profit analysis
- Regional sales trends
- Payment method analysis
- Sales prediction using Machine Learning

---

## Datasets Used

- customers.csv
- products.csv
- orders.csv
- payments.csv
- final_merged_dataset.csv

---

## Tools & Technologies

- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Power BI
- Jupyter Notebook

---

## Project Workflow

### Data Collection
Synthetic datasets created for customers, products, orders, and payments.

### Data Cleaning
- Removed duplicates  
- Handled missing values  
- Converted data types  

### Data Integration
All datasets merged using:
- CustomerID
- ProductID
- OrderID

### Feature Engineering
- TotalSales  
- DiscountAmount  
- FinalAmount  
- Profit  
- Profit Margin  

### Exploratory Data Analysis
- Sales distribution analysis  
- Category performance  
- Country-wise revenue  
- Customer segmentation  

### Machine Learning Model
- Linear Regression used for sales prediction  
- Performance evaluated using MAE, MSE, R² Score  

---

## Power BI Dashboard

A professional dashboard was created including:
- KPI Cards (Revenue, Profit, Orders, Customers)
- Monthly Sales Trend
- Category-wise Performance
- Country-wise Analysis
- Top Customers & Products
- Payment Analysis

---

## Key Insights

- Beauty and Electronics are top-performing categories  
- USA and Japan generate highest revenue  
- UPI is the most used payment method  
- Strong correlation between Price and Sales  
- Profit varies significantly across categories  

---

## Machine Learning Performance

- Model: Linear Regression  
- R² Score: ~0.86  
- Good predictive accuracy for business forecasting
